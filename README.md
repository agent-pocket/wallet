# Meme Wallet

![Banner](src/res/banner.png)

The most meme-able crypto wallet extension for Chrome.

## Features
- **Networks**: BSC, Ethereum, Arbitrum, Polygon.
- **Security**: Local encrypted storage (AES).
- **Relay**: Secure WebSocket connection for DApps.
- **Theme**: Bright Yellow Neumorphism.

## Usage
1. Load unpacked extension.
2. Create or Import Wallet.
3. Meme away!
